
# QuickBite

QuickBite is a restaurant menu discovery app that allows users to browse dishes and explore restaurant menus.

## Planned Features

1. Browse restaurant menu items
2. Search dishes by name
3. Filter menu by category
4. Save favorite dishes
5. Responsive mobile layout

## Tech Stack

- React (Vite)
- JavaScript
- CSS
- localStorage
- Netlify for deployment
